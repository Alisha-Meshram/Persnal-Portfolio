About

Hi, I'm Alisha

Some inspiring words to describe yourself

Hi, I’m Aalisha, a passionate MERN Stack Developer with a strong interest in creating modern, responsive, and user-friendly web applications. I specialize in MongoDB, Express.js, React.js, and Node.js, building full-stack projects that combine clean code with great design.

How I Got Here

My journey as a MERN Stack developer started with a curiosity for how websites and applications are built. I began learning HTML, CSS, and JavaScript, which quickly led me to explore React.js for dynamic frontends and Node.js with Express.js for backend development. Along the way, I mastered MongoDB for database management and honed my skills through real-world projects, personal experiments, and collaborative work. Each step of the journey has strengthened my problem-solving abilities, improved my coding practices, and fueled my passion for creating full-stack web applications that make a difference.

Skill

Frontend Development

I create responsive and visually appealing user interfaces using React.js, HTML5, CSS3, and JavaScript (ES6+). I focus on building interactive, user-friendly, and optimized frontends that provide seamless experiences across all devices.

Backend Development

I develop efficient and scalable server-side applications using Node.js and Express.js, ensuring smooth communication between the frontend and database while maintaining clean and reliable code.

Database Management

I work with MongoDB and Mongoose to design flexible schemas, handle data efficiently, and maintain robust, high-performance databases for every project.

Tools & Platforms

I use Git, GitHub, VS Code, and deployment platforms like Netlify, and Render to streamline development, test APIs, and deploy applications quickly and reliably.
